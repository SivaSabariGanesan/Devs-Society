Devs Society Portal – Final Ideation Draft

1. Vision & Purpose

A centralized platform to manage Devs Society operations across multiple colleges.

  • OTP-based student registration
  • Membership dashboard with digital ID card
  • Event participation (free/paid) and QR-based check-ins
  • Core Admin dashboard to manage events, users, forms, colleges, and roles

2. Portal Structure

Core Admin Portal:

  • Predefined login (email + password)
  • Add/manage Core/Admin users
  • Manage events, colleges, batches, and roles
  • QR scan for check-in

Student Portal:

  • OTP-based registration
  • Register for events, get QR & ID card
  • Manage My Events (view, countdown)
  • Online payments for paid events

3. Roles & Access

  • Core/Admin: Full access, only added manually
  • Student: Can register via OTP, limited access

4. Authentication & Registration

  • Name, email, phone number
  • Select college (REC / RIT), batch (e.g., Swayam tenure), role
  • OTP verification (email)
  • Password setup after OTP
  • Auto-generated ID (e.g., DEVS-2025-0001)
  • QR code generated

5. Student Dashboard (Post Login)

  • Dashboard: Greeting, stats
  • My Events: Registered events, QR, countdown
  • My Card: Profile info + QR
  • Logout

6. Event Management (Core/Admin)

  • Add/edit/delete events
  • Free/paid events (Cashfree/Razorpay)
  • Custom registration forms
  • View and download participant data

7. QR-Based Attendance

  • QR generated on registration
  • Scan by Core/Admin to mark attendance
  • Timestamp & duplicate prevention

8. Email Notifications

  • OTP email on signup
  • Confirmation with QR
  • Auto-reminder 1 day before event

9. Admin Panel Features

  • Add colleges, batches, roles
  • Manage Core/Admin accounts
  • Event control, stats, QR scanner

10. Tech Stack

  • Frontend: React.js + Tailwind CSS
  • Backend: Node.js + Express.js
  • Database: MongoDB
  • Auth: JWT, bcrypt, Nodemailer (OTP)
  • Payments: Razorpay, Cashfree
  • QR: qrcode, react-qr-reader
  • Reminders: node-cron

11. Advanced Dashboard Features

  • Filter by past/upcoming events
  • Countdown for events
  • QR scanner shortcut (Core/Admin only)

12. Suggested Page Structure

  • Landing Page: About, Register, Login
  • Student Dashboard: Dashboard, My Events, My Card, Logout
  • Admin Panel: Events, Users, QR Scanner, Forms, Stats
  • QR Page: Optimized for mobile scanning
